    2.4.2 文件二 services.c

    2.4.2.1 文件描述

services.c文件的完整文件名称为security/selinux/ss/services.c，该文件用于实现安全服务器，其中定义了安全服务器相关操作的实现函数。

2.4.2.3 结构体定义

该文件定义了一个名为selinux_mapping的结构体来表示内核中使用的客体类别值和策略中使用的客体类别中之间的映射关系，其代码如下所示：

 

struct selinux_mapping {

           u16 value; /* policy value */

           unsigned num_perms;

           u32 perms[sizeof(u32) * 8];

   };
