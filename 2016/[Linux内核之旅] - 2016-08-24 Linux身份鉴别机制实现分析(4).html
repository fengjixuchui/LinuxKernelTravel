2.4.1.4外部函数

1. crypt()

函数原型：char *crypt (char const *key, char const *salt);

函数功能：函数用于对输入密文进行加密。

函数参数：key表示需要加密的字符串，salt表示用的哪种加密方式。

返回值：如果成功则返回指向密文的指针，失败则返回NULL。

2. getspnam()

函数原型：struct spwd * getspnam(const char *name);

函数功能：根据登录名，返回spwd结构体指针。

函数参数：name表示登录名称。

返回值：成功返回指针，出错返回NULL。

2.4.1.5内部函数

1. xsetenv()

函数原型：static void xsetenv (char const *name, char const *val)

函数功能：用于分配空间，并查看是否内存泄漏

函数参数：其中第一个参数是用户名称。

返回值：无返回类型。

2.log_su()

其函数原型：static void log_su (struct passwd const *pw, bool successful)

函数功能：用于将整个切换过程记录在日志当中。

函数参数：pw表示传递的用户密码信息，susscessful是一个成功失败标志位。

返回值：无返回类型。

3. modify_environment()

函数原型：static void modify_environment (const struct passwd *pw, const char *shell)

函数功能：用于更改用户shell的参数。

函数参数：pw表示传递的用户密码信息，shell表示需切换到的相应shell。

返回值：无返回类型。

4. change_identity()

函数原型：static void change_identity (const struct passwd *pw)

函数功能：用于切换用户身份，更改用户的UID等信息。

函数参数：pw表示传递的用户密码信息。

返回值：无返回类型。

5. run_shell()

函数原型：static void run_shell (char const *shell, char const *command, char **additional_args,

           size_t n_additional_args)

函数功能：用于运行需切换到的用户层的shell。

函数参数：shell表示需切换到的相应shell，command表示命令行字符串。

返回值：无返回类型。

6. restricted_shell()

函数原型：static bool restricted_shell (const char *shell)

函数功能：检查切换shell是否成功。

函数参数：shell表示需切换的shell。

返回值：若成功则返回ture。

7. correct_password()

函数原型：static bool correct_password (const struct passwd *pw)

函数功能：用于检查用户输入密码是否正确。

返回值：若正确返回ture，不正确返回false。

该内部函数是su命令实现的核心部分，其流程图如下所示：

图2-7 correct_passwd流程图

1. correct_passwd()这个函数使用getspnam 读取/etc/shadow中new_user的密码，如果不为空就将获取的密码存放在correct中，如果为空则将传递的密码存放在correct中；

2. 使用getuid()获得当前用户的UID，如果UID为0表示为root用户，那么就不需要验证了，同时如果设置切换到root用户不需要密码输入的，即免密码登录也是直接切换的；

3. 用getpass()来获得用户输入密码，存放在uncrypted中；

4. 并且用crypt()函数来对这个输入密码进行加密，存放在encrypted中；

5. 使用STREQ()这个宏定义来将现在加密的密码与获取到的new_user的密码进行比较，相同则返回ture，表示验证通过。
