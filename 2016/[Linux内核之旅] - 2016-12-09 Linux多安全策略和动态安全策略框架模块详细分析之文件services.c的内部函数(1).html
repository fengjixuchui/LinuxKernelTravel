对于该文件定义的相关函数，下面进行详细解释：

①　avc_hash()

函数原型：static inline int avc_hash(u32 ssid, u32 tsid, u16 tclass)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别。

函数功能：访问向量缓存对应的哈希函数。

返回值：返回访问向量在哈希表中的哈希值。

②　avc_dump_av()

函数原型：static void avc_dump_av(struct audit_buffer *ab, u16 tclass, u32 av)；

函数参数：ab表示审计缓冲区；tclass表示客体的安全类别；av表示访问向量。

函数功能：以可读的方式显示访问向量，并根据访问向量中设置的位将相应的字符形式的权限写入审计缓冲区。

返回值：无

③　avc_dump_query()

函数原型：static void avc_dump_query(struct audit_buffer *ab, u32 ssid, u32 tsid,

 u16 tclass)；

函数参数：ab表示审计缓冲区；ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别。

函数功能：以可读的方式显示SID对和客体类别。

返回值：无。

④　avc_init()

函数原型：void __init avc_init(void)；

函数参数：无。

函数功能：初始化访问向量缓存。

返回值：无。

⑤　avc_get_hash_stats()

函数原型：int avc_get_hash_stats(char *page)；

函数参数：page指向统计信息存放的缓冲区。

函数功能：获取访问向量缓存的相关统计信息，并将相应的信息写入到参数page指定的缓冲区中。

返回值：返回写入到缓冲区中字符数。

⑥　avc_node_free()

函数原型：static void avc_node_free(struct rcu_head *rhead)；

函数参数：rhead指向待释放的访问向量结点。

函数功能：释放AVC结点在缓存中的空间。

返回值：无。

⑦　avc_node_delete()

函数原型：static void avc_node_delete(struct avc_node *node)；

函数参数：node指向待删除的访问向量结点。

函数功能：删除参数node指定的访问向量结点。

返回值：无。

⑧　avc_node_kill()

函数原型：static void avc_node_kill(struct avc_node *node)；

函数参数：node表示待释放的访问向量结点。

函数功能：释放参数node对应的缓冲区，并对访问向量缓存的相关统计信息进行修改。返回值：无。

⑨　avc_node_replace()

函数原型：static void avc_node_replace(struct avc_node *new, struct avc_node *old)；

函数参数：new表示新的访问向量结点；old表示旧的访问向量结点。

函数功能：用新的AVC结点new来替换旧的AVC结点old。

返回值：无。

⑩　avc_reclaim_node()

函数原型：static inline int avc_reclaim_node(void)；

函数参数：无。

函数功能：回收AVC_CACHE_RECLAIM个访问向量结点。

返回值：成功执行时返回回收的访问向量结点的数量。

11　avc_alloc_node()

函数原型：static struct avc_node *avc_alloc_node(void)；

函数参数：无。

函数功能：分配一个访问向量结点，并对其进行初始化。

返回值：返回分配的访问向量结点。

12　avc_node_populate()

函数原型：static void avc_node_populate(struct avc_node *node, u32 ssid, u32 tsid,

u16 tclass, struct av_decision *avd)；

函数参数：node表示待设置的访问向量结点；ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；avd表示访问向量决策。

函数功能：用指定的参数来设置访问向量缓存结点中的相应数据。

返回值：无。

13　avc_search_node()

函数原型：static inline struct avc_node *avc_search_node(u32 ssid, u32 tsid, u16 tclass)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别。

函数功能：根据源SID、目标SID、目标的类别在访问向量缓存中查找指定的结点。

返回值：如果找到，则返回相应的结点；否则返回NULL。

14　avc_lookup()

函数原型：static struct avc_node *avc_lookup(u32 ssid, u32 tsid, u16 tclass)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别。

函数功能：基于(ssid,tsid)查找访问向量。

返回值：若存在，返回包含该访问向量的结点，反之返回NULL。

15　avc_latest_notif_update()

函数原型：static int avc_latest_notif_update(int seqno, int is_insert)；

函数参数：seqno表示一个序列号；is_insert表示一个标志。

函数功能：根据参数seqno来更新访问向量缓存中最新的撤销通知序列号。

返回值：成功时返回0；反之返回相应的错误码。
