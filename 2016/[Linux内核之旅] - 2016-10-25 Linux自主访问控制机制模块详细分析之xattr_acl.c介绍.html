2.4.2 文件二 xattr_acl.c

2.4.2.1 文件描述

    对于xattr_acl.c，其完整文件名称为fs/xattr_acl.c，该文件实现了内存表示形式的ACL与扩展属性之间转换函数。

2.4.2.5 内部函数

对于该文件定义的相关函数，下面进行详细解释：

    1. posix_acl_from_xattr()

    函数原型：struct posix_acl * posix_acl_from_xattr(const void *value, size_t size)；

    函数参数：value表示扩展属性所在缓冲区；size表示缓冲区的长度。

    函数功能：从扩展属性中获取内存格式的ACL。

    返回值：成功时返回内存格式的ACL；失败时返回NULL。

    2. posix_acl_to_xattr()

    函数原型：int posix_acl_to_xattr(const struct posix_acl *acl, void *buffer, size_t size)；

    函数参数：acl表示内存形式的访问控制列表；buffer表示存放扩展属性的缓冲区；size表示缓冲区的长度。

    函数功能：将内存表示形式的ACL转换为磁盘表示形式，并将其存放到扩展属性中。

    返回值：成功执行时返回扩展属性的长度；失败时返回相应的错误码。

2 posix_acl_to_xattr()

int posix_acl_to_xattr(const struct posix_acl *acl, void *buffer, size_t size)

{

        posix_acl_xattr_header *ext_acl = (posix_acl_xattr_header *)buffer;

        posix_acl_xattr_entry *ext_entry = ext_acl->a_entries;

        int real_size, n;

/*

 * 获取含有指定数量的ACL表项的访问控制列表在扩展属性中占有的存储空 * 间的大小 

 */

        real_size = posix_acl_xattr_size(acl->a_count);

        if (!buffer)

                return real_size;

        if (real_size > size)

                return -ERANGE;

        /*

 * 将一个32位的特定于cpu格式的用于表示ACL的a_version域的数据转化 * 为相应的小端格式 

 */

        ext_acl->a_version = cpu_to_le32(POSIX_ACL_XATTR_VERSION);

/*

 * 通过循环将ACL中每个ACL表项中的cpu格式的e_tag、e_perm和e_id * 转换为相应的小端格式，并存放到指定的扩展属性中

 */

        for (n=0; n < acl->a_count; n++, ext_entry++) {

                ext_entry->e_tag  = cpu_to_le16(acl->a_entries[n].e_tag);

                ext_entry->e_perm = cpu_to_le16(acl->a_entries[n].e_perm);

                ext_entry->e_id   = cpu_to_le32(acl->a_entries[n].e_id);

        }

        return real_size;

}
