洪峰简介：

　　自由软件运动倡导者，2001 年开创了“一、百、万”工程。“明”操作系统的设计师、泛系“黑客道”教学法的创始人。擅长教授关于哲学、数学、计算机科学、经济学、语言学等多方面的课程。

 人生使命：人生一世，其价值如何才能得到体现？ 如何生活和工作，才能使自己的价值最大化？ 那就是你最好创造出一些有用的东西留给世界。换句话说，你应该努力成为一名“创客”（Maker）。成为创客之前，你需要接受系统的训练，使自己得到创客必备的创新技能。成才法，是黑客道关注、研究、实践和推广的内容。


在前面的第四讲中我给大家介绍了面粉与石膏，也就是用数学知识进行建模。以及根据数 学模型建立计算模型。一旦有了针对一个特性专题的计算模型之后，接下来一个很重要的任务是把计算模型转换为计算机程序。让计算机程序可以在计算机上运行，在这个过程中我们必须用某种或者某些编程语言来实现这个目标。
那么在这一讲，我们谈一谈计算机语言。
当我们接受这个问题的时候，马上就有一个非常困难的局面摆在我们面前。因为迄今为止，计算机科学家们发明出来的计算机语言种类非常多。我们人的时间，生命是有限的。我们不可能掌握太多太多的编程语言。这样呢，就有一个选择的问题。那么怎么样选择适当的编程语言，去实现我们的计算模型。
这里我来谈谈我的一些看法。在往下进行之前，我想首先摆明一下，我选择编程语言的原则。
这里把它归纳为三点。第一，我不会去向大家介绍一些已经死掉的一些语言。语言跟人生命一样，有生老病死。历史上有很多语言被发明出来，在历史上曾经流行过、辉煌过。甚至在某段时间占据了统治地位。

但是随着时代的发展，这些语言依存的环境消失了。利用这些语言开发程序的群体消失了，所以这些语言退出了历史的舞台。所以在以后的介绍里面，我不会去触及这一类语言；那么呢，第二个原则是，我不会向大家介绍一些专有的编程语言。所谓专有的计算机编程语言，就是指大家需要向某个商业公司购买专有的许可证，这样的语言。这样的语言也非常多，我在这里不会触及这类的语言；那么第三个，我不会向大家介绍一些只能用于专有系统的编程语言。比方说，微软公司的Microsoft Windows是个专有系统。它上面有很多语言，比如：C#。它可能设计的非常好，但是它只能工作在这些专有系统上。那么这样的一些语言，我在后面的介绍中不会去触及它。

我之所以确立上面三条选择语言的原则，我觉得让大家选择自由开放的计算机编程语言进行创业活动。对于大家减少成本、增加成功的机会非常有帮助的。
我们中国人有句古话，叫做：兼听则明偏信则暗。我认为这个方法，运用在我们讨论计算机语言的选择的时候是很有帮助的。我们不妨来看一看世界上其他的编程大师们，他们是怎么样选择编程语言，他们的观点是怎么样的。这里呢，第一位要向大家介绍的是美国的一位著名的黑客，他的名字叫做Eric Raymond 。他写过很多程序，包括处理终端的Ncurses这个库是他写的。还有一个很有名的客户端的邮件收发程序Fetchmail，也是他写的。而且Eric Raymond是一位很好的作家。他写过很多散文，还有很多的书。比较有名的书包括 《黑客简史》、《教堂与集市》。英文叫做《The Cathedral and the Bazaar》。这个书的影响是非常大的，大家有时间可以去看一看。那么他曾经写过一篇很有名的文章，叫做《How to Become a Hacker》《怎么样成为一个黑客》。在这篇文章里面，他提了一个观点，就是初学者，可以从Python、Java开始学习计算机编程。然后，认真的、严肃的黑客会学习 C。因为你要在Unix系统上工作，你必须要用 C 。 C 是Unix系统的母语。然后呢，会选择Perl来搞系统管理。最后呢，真正认真的黑客应该学习Lisp。他呢，提出了5种黑客要掌握的语言。那就是Python、Java、C、Perl、Lisp。这是他的一个观点。

接下来呢，我们来看看美国的计算机语言大师 Larry Wall先生。他是怎么来看待这个问题的。Larry Wall是Perl语言的发明人。他曾经呢，制作过一段视频。谈谈每一位学习计算机的人应该知道哪些编程语言。他认为首先应该从最流行的Javascript开始。因为Javascript现在已经内置到每一台计算机的浏览器里面去了。每一个浏览器里面都有，现代的浏览器都有Javascript的实现。Javascript是世界上的目前最流行的编程语言。其次呢，他认为应该学习Java。Java现在企业级应用里面非常普遍。第三种他推荐的语言呢，是Haskell。Haskell这个语言呢，我们后面的内容还会反复提到。他认为Haskell是Lisp的现代版本。第四种他推荐的语言是 C ，搞系统编程。最后呢，他推荐使用一种脚本语言。可以从Perl、Python、Ruby这些脚本语言中去选一。当然呢，他因为是Perl语言的发明人，他当然会推荐Perl。所以Javascript、Java、Haskell、C、Perl是Larry Wall先生推荐的五种编程语言。

我这里要向大家介绍的第三位黑客是我的朋友，Richard Stallman 。他是自由软件基金会的总裁，也是 GNU project的发起人。他的作品很多，最有名的包括 GNU Emacs文本编辑器，还有 GCC 等等。我曾经写信问过 Stallman院士 Stallman是美国工程院院士。我问，我说黑客应该掌握哪些语言。他曾经回复我，这样讲的，他说刚开始的时候，应该从C开始。先学C。而一旦你知道了C之后，那么呢，你需要掌握一种高级的、具有非常表达能力的编程语言。而这种语言非Lisp莫属。 最后真正的黑客要发挥计算机系统的硬件特性，他一定要知道计算机的工作原理、工作流程。要达到这个目的呢，计算机黑客要知道计算机汇编语言怎么去用。最后呢 Stallman院士呢他推荐 c 、Lisp 和 汇编 三种语言。在刚才Stallman院士提到的三种编程语言，也就是 C 、Lisp 和 汇编 中，C、Lisp属于高级语言，汇编语言属于低级汇编语言。在整个高级语言的谱系里面，C 和 Lisp处在两个极端。在两个极端之中呢，又有很多其他的那些编程语言。关于这个观点的细节呢，我们以后专门找时间 展开和说明。
C语言呢，可以非常好的用于硬件抽象，而Lisp语言可以非常好的用于算法的抽象和表达。所以我把这一章的题目定为《字符与球拍》，它的寓意也是从这个地方受到启发得来。C语言呢，非常擅长对于字符进行处理。所以在计算机工业界内部，我们有一个戏称。把C语言称之为字符处理器，叫《Character Process》。而Lisp是由list process列表处理器这个词语撰写而来的。所以C和Lisp代表了高级编程语言的两个高峰，这个观点是得到业内一致认可的。

最后我向大家介绍一位美国的黑客，非常有名，他的名字叫做Paul Graham。他曾经和他的同学，也是MIT计算机系的一个教授，叫做Morris，Robber Morris。他们两人在九十年代中期合作开发了第一个Web上的商业应用，叫做网络商店。这个网络商店名字叫做Viaweb，后来被雅虎公司收购。在这个项目里面呢，Paul Graham他利用了Common Lisp打造领域专用语言。而Robber Morris负责定制各种模板。他们两人在这个项目里面把Common Lisp和Perl两个语言呢巧妙的结合起来，获得了很大成功。这个Paul Graham和Robber Morris把公司卖给雅虎以后呢，过段时间他们又用这笔钱去做风险投资，就是IT领域的风险投资。从2005年到2015年，这十多年的时间内呢，他们先后资助了一千多家IT领域的创新公司。取得了非常大的成功，这在硅谷也堪称奇迹。Paul Graham曾经写过一本书《Hackers and Painters》，中文叫做《黑客与画家》。因为保罗，他本人也是一个画家，曾经学过绘画，具有艺术气质的画家。关于他们当年Viaweb项目开发的详细情况呢，在这个书中也有很多披露。有兴趣的同学呢，可以去看一看。
Stallman院士曾经还跟我补充过一个观点，对于编程语言的选择，有一半的因素属于技术性的考虑，另外一般因素来自于黑客个人的喜好、偏好。所以呢，我觉得他的这个观点也是非常有价值的，值得大家参考。

前面呢，我已经先后向大家介绍了四位大师级的黑客，还有专家学者。他们对于计算机语言编程选择的观点，大家都已经看到，他们的观点都是不完全相近的。那么有同学，有听众会问，洪老师。你的观点是什么。在这里呢，我要借这个机会。向大家提供我的观点，说明我的观点。我的观点呢，跟Stallman院士的观点有很大的相似性。我认为要从系统编程和算法构造，对于硬件的抽象和算法构造两个层面进行基本功的修炼。
但是真正对于编程，这个项目里面，对于创客，创造项目里面。我有一个观点就是，你必须针对你所创业的项目的任务的特点，来选择某一种、某一些编程语言来进行开发。这是我的观点。下面呢，我会花更多的时间，把我的观点详细的展开和说明，且听下文的分解。
