16　avc_insert()

函数原型：static struct avc_node *avc_insert(u32 ssid, u32 tsid, u16 tclass,

  struct av_decision *avd)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；avd表示访问向量决策。

函数功能：根据指定的源SID、目标SID、客体类别向访问向量缓存中插入一个访问向量。

返回值：成功时返回插入的访问向量结点；反之返回NULL。

17　avc_audit_pre_callback()

函数原型：static void avc_audit_pre_callback(struct audit_buffer *ab, void *a)；

函数参数：ab表示审计缓冲区；a指向用于生成审计记录的辅助数据。

函数功能：通用审计代码调用的回调函数，用于获取SELinux指定的信息。

返回值：无。

18　avc_audit_post_callback()

函数原型：static void avc_audit_post_callback(struct audit_buffer *ab, void *a)；

函数参数：ab表示审计缓冲区；a指向用于生成审计记录的辅助数据。

函数功能：通用审计代码调用的回调函数，用于获取SELinux指定的信息。

返回值：无。

19　slow_avc_audit()

函数原型：noinline int slow_avc_audit(u32 ssid, u32 tsid, u16 tclass,u32 requested,

 u32 audited, u32 denied,

struct common_audit_data *a,unsigned flags)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；requested表示请求的权限；audited表示待审计的权限；denied表示拒绝的权限；a表示用于生成审计记录的辅助数据；flags表示文件的打开标志。

函数功能：用于AVC的审计。

返回值：成功时返回0；反之返回相应的错误码。

20　avc_add_callback()

函数原型：int __init avc_add_callback(int (*callback)(u32 event), u32 events)；

函数参数：callback表示事件对应的回调函数；events表示待处理的事件。

函数功能：用于为event中指定的事件注册一个回调函数，并将该函数插入到回调函数链表中。

返回值：成功时，返回0；反之返回相应的错误码。

21　avc_sidcmp()

函数原型：static inline int avc_sidcmp(u32 x, u32 y)；

函数参数：x，y分别表示待比较的SID。

函数功能：判断两个SID是否相等。

返回值：若相等或其中一个为SECSID_WILD，则返回真。

22　avc_update_node()

函数原型：static int avc_update_node(u32 event, u32 perms, u32 ssid, u32 tsid, u16 tclass,

u32 seqno)；

函数参数：event表示更新事件；perms表示相应的权限掩码；ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；seqno表示生成访问向量决策使用的序列号。

函数功能：更新访问向量缓存中的访问向量。如果指定的访问向量存在，则更新该访问向量，并释放之前的访问向量。

返回值：成功时返回0；反之返回相应的错误码。

23　avc_flush()

函数原型：static void avc_flush(void)；

函数参数：无。

函数功能：刷新AVC，该函数实际上调用了avc_node_delete()函数来删除访问向量缓存中所有的AVC结点。

返回值：无。

24　avc_ss_reset()

函数原型：int avc_ss_reset(u32 seqno)；

函数参数：seqno表示策略中的序列号。

函数功能：刷新访问向量缓存，并调用回调函数重新使能迁移的权限。

返回值：成功时返回0；反之返回相应的错误码。

25　avc_compute_av()

函数原型：static noinline struct avc_node *avc_compute_av(u32 ssid, u32 tsid,u16 tclass,     struct av_decision *avd)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；avd表示相应的访问向量决策。

函数功能：通过调用安全服务器中的security_compute_av()函数来计算一个访问向量决策，并将相应的访问向量插入到访问向量缓存中。

返回值：返回插入到访问向量缓存中结点。

26　avc_denied()

函数原型：static noinline int avc_denied(u32 ssid, u32 tsid,u16 tclass, u32 requested,

  unsigned flags,struct av_decision *avd)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；requested表示请求的权限；flags表示一个标志，该标志用于判定是否忽略相应的权限位；avd表示相应的访问向量决策。

函数功能：该函数实际上用于为参数指定的访问向量授予requested指定的权限。

返回值：成功时返回0；否则返回相应的错误码。

27　avc_has_perm_noaudit()

函数原型：inline int avc_has_perm_noaudit(u32 ssid, u32 tsid,u16 tclass, u32 requested,

                          unsigned flags,struct av_decision *avd)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；requested表示请求的权限；flags表示一个用于判定是否忽略相应的权限位标志；avd表示相应的访问向量决策。

函数功能：在不执行审计的情况下检查访问向量缓存来确定是否授予SID对(ssid,tsid)以请求的权限，如果在AVC中没有找到指定的决策，则查询安全服务器，并将相应的决策插入到AVC中。

返回值：如果授予了指定的权限，则返回0；反之返回相应的错误码。

28　avc_has_perm_flags()

函数原型：int avc_has_perm_flags(u32 ssid, u32 tsid, u16 tclass,u32 requested,

  struct common_audit_data *auditdata,unsigned flags)；

函数参数：ssid表示主体的SID；tsid表示客体的SID；tclass表示客体的类别；requested表示请求的权限；auditdata表示用于生成设计记录的辅助数据；flags表示VFS的处理标志。

函数功能：检查访问向量缓冲来确定是否授予了SID对(ssid,tsid)以请求的权限，并根据策略审计请求的权限。

返回值：如果授予了指定的权限，则返回0；反之返回相应的错误码。

29　avc_policy_seqno()

函数原型：u32 avc_policy_seqno(void)；

函数参数：无。

函数功能：获取访问向量缓存的最新的撤销通知序列号。

返回值：返回访问向量缓存的序列号。

30　avc_disable()

函数原型：void avc_disable(void)；

函数参数：无。

函数功能：禁止AVC。

返回值：无。
