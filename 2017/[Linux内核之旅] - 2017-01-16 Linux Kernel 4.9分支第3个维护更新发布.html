Linux Kernel 4.9分支第3个维护更新发布

作者：嵌入式Linux 中文站

本文转载自嵌入式Linux中文站

和上个版本相隔仅仅三天时间，Linux内核维护人员Greg Kroah-Hartman就宣布了Linux Kernel 4.9分支的第三个维护版本更新。根据短日志，本次更新共对201份文件进行调整，插入1929处，删除945处。


Greg Kroah-Hartman说道：“Linux Kernel 4.9.3正式发布。所有Linux 4.9分支的用户必须尽快完成升级。已经升级的4.9.y tree可以在 git://git.kernel.org/pub/scm/linux/kernel/git/stable/linux-stable.git linux-4.9.y这里找到，此外，用户也可以通过网页浏览器访问Kernel.org访问，地址为http://git.kernel.org/?p=linux/kernel/git/stable/linux-stable.git;a=summary,”


Linux Kernel 4.9.3维护更新对XFS文件系统带来了大量改善，对USB, wireless (ath10k, rtlwifi), IOMMU, iiO, hwmon, Intel i915, CLK，部分MMC, PINCTRL, PCI, SCSI和IRQ Chip进行了驱动升级，对ARM, ARM64/AArch64, MIPS, x86, CRIS, s390和PA-RISC架构进行了调整。


此外Linux Kernel 4.9.3还对mac80211, wireless (nl80211), Intel Skylake, Intel HDA以及Samsung I2S的声音和网络堆栈进行了调整。这是自2016年12月11日Linux Kernel 4.9正式版本发布以来，最大的维护版本更新。
