5.crypto_destroy_tfm

函数原型：void crypto_destroy_tfm(void *mem, struct crypto_tfm *tfm);

函数功能：算法对象的销毁。

函数参数：mem 内存空间；tfm 算法对象。

函数返回值：无。

执行流程：

（1）判断传入参数mem为空的可能性，如果不为空的可能性比较大，则直接返回。

（2）通过算法对象获取算法alg。

（3）判断对象是否退出且算法是否退出，如果算法退出而对象没有退出，调用算法退出算法对象。

（4）后续处理工作，包括算法退出操作、模块释放以及内存空间的释放。

6.crypto_has_alg

函数原型：int crypto_has_alg(const char *name, u32 type, u32 mask);

函数功能：算法是否存在。

函数参数：name 算法名称；type 算法类型； mask 算法掩码。

函数返回值：返回0表示算法存在。

执行流程：调用函数crypto_alg_mod_lookup获得算法alg，如果获取成功，则释放算法模块并令ret等于1，然后返回ret变量。
